How it works

What is directional overcurrent?

The relay compares the angle of an operating current phasor I (or I + k·I₀ for earth) to a stable voltage reference Vpol. If the angle of I relative to Vpol falls inside the forward operating semicircle (here drawn as ±90° about MTA), the element declares “forward” and can trip; otherwise it restrains.

MTA = ref_angle + MTA_offset (mod 360°)

Phase–phase (cross-polarization)

For phase-to-phase faults, polarization uses a line-to-line voltage that avoids the most collapsed phase where possible (e.g. R-related loop → VYB). Healthy phases still provide a measurable reference when the faulted phase voltage is small, so direction stays reliable near the relay.

Earth fault (zero-sequence polarization)

For earth faults, the reference aligns with zero-sequence voltage 3V₀ (labeled V₀(R), etc. in this demo). Residual / zero-sequence current I₀ is compared to V₀: V₀ rises when the network’s neutral point shifts during ground faults, giving a consistent directional reference.

MTA and blind zone

MTA (maximum torque axis) is the centerline of the comparator’s sensitive direction. Blind zones are narrow wedges straddling the ±90° boundaries to reduce chatter when the measured angle sits exactly on the edge.

Forward vs Reverse

Forward uses MTA = ref + offset. Reverse shifts MTA by 180° so you visualize the reverse-directional characteristic: the green operate band rotates with MTA on the phasor diagram.
